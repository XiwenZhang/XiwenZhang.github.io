The perfect Bootstrap 4 admin template for app developersOur application is aims to achieve could-driven AR/VR technology is integrated into collaboration and immersive experience for uses in the smart home content and use the upcoming 5G network technology and processing and analysis of spatial data to complete real-time 3D model creation.In the application work flow in smart home content:

New

1. Users upload depth information of their object view into Could services.

2. Could algorithms will use specific algorithms to process information and split the view to create a lightweight 3D models by converting point-clouds into polygons and record the key elements with its 3D location information.

3. Processing and completing as a complete 3D model from the recorded elements and lightweight 3D models.
